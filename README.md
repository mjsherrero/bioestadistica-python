# Applied Biostatistics with Python

Educational repository focused on introductory biostatistics and quantitative data analysis applied to health sciences.

## Topics covered
- Descriptive statistics
- Hypothesis testing
- Correlation analysis
- Linear regression
- Data visualization

## Tools
- Python
- pandas
- scipy
- matplotlib
- statsmodels
- Jupyter Notebook

## Educational approach
The notebooks are designed for students in:
- Nursing
- Psychology
- Physiotherapy
- Health sciences

## Repository structure

```text
datasets/
notebooks/
figures/
```

## Current development
This repository is currently under development and will include practical examples and educational notebooks for applied biostatistics.
